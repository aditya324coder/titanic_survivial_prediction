


📌 # titanic_survivial_prediction (Exploratory Data Analysis)

This project performs Exploratory Data Analysis (EDA) on the famous Titanic dataset using Python. The goal is to analyze passenger data and identify patterns that influenced survival during the Titanic disaster.

🚀 Features
Data loading using Seaborn’s built-in Titanic dataset
Data cleaning and preprocessing:
Removal of irrelevant columns (deck, embark_town, etc.)
Handling missing values (e.g., filling age with mean)
Statistical summary and dataset inspection
Visualization-ready setup using Matplotlib and Seaborn
🛠️ Tech Stack
Python
NumPy
Pandas
Seaborn
Matplotlib
📊 Key Objectives
Understand dataset structure and features
Handle missing and unnecessary data
Prepare dataset for further analysis or machine learning
Discover survival trends based on passenger attributes
📁 Dataset

The dataset is loaded using:

sns.load_dataset('titanic')
📌 Conclusion

This project lays the foundation for data preprocessing and visualization, which can be extended further into machine learning models for survival prediction.
